# Product Images

This folder contains product images for KOSE products used in the mock data.

## Current Status

Placeholder images are being used. To use real KOSE product images:

1. Visit https://www.kose-th.com/
2. Navigate to product pages
3. Download product images
4. Save them with the following filenames:

- `sekkisei-lotion.jpg` - Sekkisei Clear Wellness Lotion
- `sekkisei-emulsion.jpg` - Sekkisei Clear Wellness Emulsion
- `sekkisei-wash.jpg` - Sekkisei Clear Wellness Wash
- `decorte-cream.jpg` - Decorte AQ Meliority Cream
- `decorte-serum.jpg` - Decorte AQ Meliority Serum
- `infinity-lotion.jpg` - Infinity Pure Moisture Lotion
- `infinity-emulsion.jpg` - Infinity Pure Moisture Emulsion
- `esprique-cream.jpg` - Esprique Precious Rich Cream
- `onebykose-mask.jpg` - One by Kose Clear Turn Mask
- `sekkisei-powder-wash.jpg` - Sekkisei White Powder Wash

## Image Specifications

- Format: JPG or PNG
- Recommended size: 300x300px or larger
- Aspect ratio: Square (1:1)
- File size: Keep under 500KB per image for optimal performance

## Usage

Images are referenced in `src/mock/product.ts` using paths like `/products/sekkisei-lotion.jpg`
